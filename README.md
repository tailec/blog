# blog
Visit [www.tailec.com](http://www.tailec.com)

## Setup
```shell
gem install jekyll
gem install glynn
```

## Run
```shell
jekyll s
```
Go to `127.0.0.1:4000`

## Production
Create your `_keys.yml`
```shell
JEKYLL_ENV=production jekyll b -c _config.yml,_keys.yml
```
